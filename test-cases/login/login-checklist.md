# Login Checklist

## UI
- Login page loads successfully
- Username field is visible
- Password field is visible
- Login button is visible and enabled

## Functional
- Login with valid credentials
- Login with invalid password
- Login with invalid username
- Login with empty username
- Login with empty password
- Login with both fields empty

## Validation
- Error message for empty username
- Error message for empty password
- Error message for invalid credentials
- Error messages disappear after correction

## Security (basic)
- Password field hides characters
- Login form does not allow HTML tags
- Login form does not allow SQL injection patterns

## Navigation
- User is redirected to dashboard after successful login
- User remains on login page after failed login
