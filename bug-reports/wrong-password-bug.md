# Bug Report: Login with wrong password

## Description
When user enters a wrong password, the error message is unclear.

## Steps to reproduce
1. Open login page
2. Enter valid username
3. Enter wrong password
4. Click "Login" button

## Expected result
Clear error message is displayed (e.g. "Incorrect password")

## Actual result
Error message is vague or confusing (e.g. "Invalid credentials")

## Severity
Low
