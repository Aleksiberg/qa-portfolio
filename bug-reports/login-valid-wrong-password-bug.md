# Bug Report: Login with wrong password

## Description
When a user enters an incorrect password, the system displays a generic error message ("Invalid credentials") instead of a specific message indicating that the password is incorrect.

Environment:
- Browser: Chrome 120
- OS: Windows 11

## Steps to reproduce
1. Open login page
2. Enter valid username
3. Enter wrong password
4. Click "Login" button

## Expected result
System should display a clear and specific error message such as "Incorrect password" to help the user understand the issue.

## Actual result
Error message is vague or confusing ("Invalid credentials")

## Severity
Low

## Priority
Medium
