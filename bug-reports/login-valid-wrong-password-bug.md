# Bug Report: Login with wrong password

## Description
When user enters a wrong password, the error message is unclear.

Environment:
- Browser: Chrome 120
- OS: Windows 10

## Steps to reproduce
1. Open login page
2. Enter valid username
3. Enter wrong password
4. Click "Login" button

## Expected result
Error message is displayed ("Incorrect password")

## Actual result
Error message is vague or confusing ("Invalid credentials")

## Severity
Low

## Priority
Medium
