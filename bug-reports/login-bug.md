# Bug Report: Login with empty fields

## Description
When user tries to log in without entering username and password, no error message is shown.

## Steps to reproduce
1. Open login page
2. Leave username field empty
3. Leave password field empty
4. Click "Login"

## Expected result
Error message is displayed (e.g. "Username and password are required")

## Actual result
Nothing happens, no error message is shown

## Severity
Medium
