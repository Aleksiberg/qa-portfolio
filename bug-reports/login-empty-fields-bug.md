# Bug Report: No validation message when submitting empty login form

## Description
Login button does not show validation error when both fields are empty

## Steps to reproduce
1. Open login page
2. Leave username empty
3. Leave password empty
4. Click "Login" button

## Expected result
The system displays: "Username and password are required" and the user remains on the login page.

## Actual result
Nothing happens after clicking the login button

## Severity
Medium

## Priority
High
